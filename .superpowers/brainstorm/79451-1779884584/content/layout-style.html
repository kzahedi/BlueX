Light theme, more breathing room, card-based posts. Colour-coded badges for hate/counter/neutral. Feels like a modern macOS app. Easier on the eyes for long reading sessions.

Both options show the same structure: sidebar (groups + accounts) → post list → thread detail with colour-coded annotation badges. The real posts and replies are in German — that's intentional.
